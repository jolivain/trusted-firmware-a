TFA_ARCH_VERSION
================

.. default-domain:: cmake

.. variable:: TFA_ARCH_VERSION

The version of the Arm architecture to target, in string format. This takes the
form of the branded architecture name, e.g. ``Armv8-A``, ``Armv8.6-A``, etc.
