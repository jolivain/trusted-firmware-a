TFA_CTX_INCLUDE_NEVE_REGS
=========================

.. default-domain:: cmake

.. variable:: TFA_CTX_INCLUDE_NEVE_REGS

Causes the Armv8.4-A nested virtualization registers to be saved/restored when
entering/exiting an EL2 execution context.

Disabled by default.
