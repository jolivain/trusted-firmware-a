TFA_ERRATA_A57_826977
=====================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_A57_826977

This applies errata 826977 workaround to the Cortex-A57 CPU. This needs to be
enabled only for revision <= r1p1 of the CPU.
