TFA_ERRATA_A78_1688305
======================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_A78_1688305

This applies errata 1688305 workaround to the Cortex-A78 CPU. This needs to
be enabled only for revision r0p0 to r1p0 of the CPU.
