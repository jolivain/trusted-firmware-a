TFA_ERRATA_A57_814670
=====================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_A57_814670

This applies errata 814670 workaround to the Cortex-A57 CPU. This needs to be
enabled only for revision r0p0 of the CPU.
