TFA_ERRATA_A75_790748
=====================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_A75_790748

This applies errata 790748 workaround to the Cortex-A75 CPU. This needs to
be enabled only for revision r0p0 of the CPU.
