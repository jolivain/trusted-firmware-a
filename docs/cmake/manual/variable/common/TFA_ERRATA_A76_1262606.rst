TFA_ERRATA_A76_1262606
======================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_A76_1262606

This applies errata 1262606 workaround to the Cortex-A76 CPU. This needs to
be enabled only for revision <= r3p0 of the CPU.
