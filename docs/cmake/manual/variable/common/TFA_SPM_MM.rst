TFA_SPM_MM
==========

.. default-domain:: cmake

.. variable:: TFA_SPM_MM

Enable the Management Mode (MM)-based Secure Partition Manager (SPM)
implementation.

Disabled by default. Forcibly disabled if using the SPM Dispatcher, or if
EL3 exceptions are disabled.
