TFA_ARM_SPMC_MANIFEST_DTS
=========================

.. default-domain:: cmake

.. variable:: TFA_ARM_SPMC_MANIFEST_DTS

Specifies the path to an alternate manifest file used as the |SPMC| manifest.
Valid when ``SPD=spmd`` is selected.
