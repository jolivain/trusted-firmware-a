TFA_XLAT_TABLES_LIB_V1
======================

.. default-domain:: cmake

.. variable:: TFA_XLAT_TABLES_LIB_V1

Compiles TF-A with version 1 of the translation tables library instead of
version 2.

Disabled by default.
