TFA_ARCH_STATE
==============

.. default-domain:: cmake

.. variable:: TFA_ARCH_STATE

Defines the target instruction set state to target, either ``AArch64`` or
``AArch32``. Note that when targeting Armv7-A, this is always ``AArch32``.
