TFA_FVP_USE_GIC_DRIVER
======================

.. default-domain:: cmake

.. variable:: TFA_FVP_USE_GIC_DRIVER

Selects the GIC driver to use. Options:

-  ``GICv2``: The GICv2 driver is selected
-  ``GICv3``: The GICv3 driver is selected

Defaults to ``GICv3``.
