TFA_ERRATA_A76_1130799
======================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_A76_1130799

This applies errata 1130799 workaround to the Cortex-A76 CPU. This needs to
be enabled only for revision <= r2p0 of the CPU.
