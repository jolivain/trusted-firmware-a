TFA_ERRATA_N1_1275112
=====================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_N1_1275112

This applies errata 1275112 workaround to the Neoverse-N1 CPU. This needs to
be enabled only for revision <= r3p0 of the CPU.
