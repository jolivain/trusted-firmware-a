TFA_ERRATA_N1_1257314
=====================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_N1_1257314

This applies errata 1257314 workaround to the Neoverse-N1 CPU. This needs to
be enabled only for revision <= r3p0 of the CPU.
