TFA_ERRATA_A9_794073
====================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_A9_794073

This applies the erratum 794073 workaround to the Cortex-A9 CPU. This needs
to be enabled for all revisions of this CPU.
