TFA_ERRATA_N1_1220197
=====================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_N1_1220197

This applies errata 1220197 workaround to the Neoverse-N1 CPU. This needs to
be enabled only for revision <= r2p0 of the CPU.
