TFA_ERRATA_N1_1542419
=====================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_N1_1542419

This applies errata 1542419 workaround to the Neoverse-N1 CPU. This needs to
be enabled only for revisions r3p0 to r4p0 of the CPU.
