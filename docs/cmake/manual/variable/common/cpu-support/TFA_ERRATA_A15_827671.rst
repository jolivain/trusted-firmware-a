TFA_ERRATA_A15_827671
=====================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_A15_827671

This applies errata 827671 workaround to the Cortex-A15 CPU. This needs to be
enabled only for revision >= r3p0 of the CPU.
