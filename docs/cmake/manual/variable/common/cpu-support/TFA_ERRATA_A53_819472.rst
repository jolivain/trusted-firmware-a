TFA_ERRATA_A53_819472
=====================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_A53_819472

This applies errata 819472 workaround to all CPUs. This needs to be enabled
only for revision <= r0p1 of the Cortex-A53.
