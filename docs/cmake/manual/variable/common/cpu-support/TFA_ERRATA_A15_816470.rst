TFA_ERRATA_A15_816470
=====================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_A15_816470

This applies errata 816470 workaround to the Cortex-A15 CPU. This needs to be
enabled only for revision >= r3p0 of the CPU.
