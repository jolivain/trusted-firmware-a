TFA_ERRATA_A53_826319
============================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_A53_826319

This applies errata 826319 workaround to the Cortex-A53 CPU. This needs to be
enabled only for revision <= r0p2 of the CPU.
