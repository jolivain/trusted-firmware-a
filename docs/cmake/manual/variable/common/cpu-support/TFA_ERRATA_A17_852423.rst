TFA_ERRATA_A17_852423
=====================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_A17_852423

This applies errata 852423 workaround to the Cortex-A17 CPU. This needs to be
enabled only for revision <= r1p2 of the CPU.
