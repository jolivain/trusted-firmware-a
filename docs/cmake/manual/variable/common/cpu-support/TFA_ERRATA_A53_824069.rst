TFA_ERRATA_A53_824069
=====================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_A53_824069

This applies errata 824069 workaround to all CPUs. This needs to be enabled
only for revision <= r0p2 of the Cortex-A53.
