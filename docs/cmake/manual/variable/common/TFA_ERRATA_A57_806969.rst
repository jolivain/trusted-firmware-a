TFA_ERRATA_A57_806969
=====================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_A57_806969

This applies errata 806969 workaround to the Cortex-A57 CPU. This needs to be
enabled only for revision r0p0 of the CPU.
