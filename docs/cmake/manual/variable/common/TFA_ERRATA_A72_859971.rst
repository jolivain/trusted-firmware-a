TFA_ERRATA_A72_859971
=====================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_A72_859971

This applies errata 859971 workaround to the Cortex-A72 CPU. This needs to be
enabled only for revision <= r0p3 of the CPU.
