TFA_USE_SP804_TIMER
===================

.. default-domain:: cmake

.. variable:: TFA_USE_SP804_TIMER

Enable usage of the SP804 timer instead of the Generic Timer for functions that wait for
an arbitrary time length (``udelay`` and ``mdelay``).

Disabled by default.
