TFA_ERRATA_A77_1925769
======================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_A77_1925769

This applies errata 1925769 workaround to the Cortex-A77 CPU. This needs to
be enabled only for revision <= r1p1 of the CPU.
