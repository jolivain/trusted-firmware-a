TFA_ERRATA_A57_859972
=====================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_A57_859972

This applies errata 859972 workaround to the Cortex-A57 CPU. This needs to be
enabled only for revision <= r1p3 of the CPU.
