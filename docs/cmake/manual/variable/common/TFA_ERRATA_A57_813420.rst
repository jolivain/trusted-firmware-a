TFA_ERRATA_A57_813420
=====================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_A57_813420

This applies errata 813420 workaround to the Cortex-A57 CPU. This needs to be
enabled only for revision r0p0 of the CPU.
