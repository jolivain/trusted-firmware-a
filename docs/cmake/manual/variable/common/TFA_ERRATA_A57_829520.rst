TFA_ERRATA_A57_829520
=====================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_A57_829520

This applies errata 829520 workaround to the Cortex-A57 CPU. This needs to be
enabled only for revision <= r1p2 of the CPU.
