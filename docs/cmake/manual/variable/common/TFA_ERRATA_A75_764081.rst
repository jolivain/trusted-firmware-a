TFA_ERRATA_A75_764081
=====================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_A75_764081

This applies errata 764081 workaround to the Cortex-A75 CPU. This needs to be
enabled only for revision r0p0 of the CPU.
