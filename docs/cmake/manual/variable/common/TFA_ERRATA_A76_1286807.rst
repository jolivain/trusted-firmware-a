TFA_ERRATA_A76_1286807
======================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_A76_1286807

This applies errata 1286807 workaround to the Cortex-A76 CPU. This needs to
be enabled only for revision <= r3p0 of the CPU.
