TFA_ERRATA_A55_798797
=====================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_A55_798797

This applies errata 798797 workaround to the Cortex-A55 CPU. This needs to be
enabled only for revision r0p0 of the CPU.
