TFA_ERRATA_A57_833471
=====================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_A57_833471

This applies errata 833471 workaround to the Cortex-A57 CPU. This needs to be
enabled only for revision <= r1p2 of the CPU.
