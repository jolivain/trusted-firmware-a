TFA_ERRATA_A76_1220197
======================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_A76_1220197

This applies errata 1220197 workaround to the Cortex-A76 CPU. This needs to
be enabled only for revision <= r2p0 of the CPU.
