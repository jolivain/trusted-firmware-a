TFA_ERRATA_N1_1315703
=====================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_N1_1315703

This applies errata 1315703 workaround to the Neoverse-N1 CPU. This needs to
be enabled only for revisions <= r3p0 of the CPU.
