TFA_ERRATA_A73_855423
=====================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_A73_855423

This applies errata 855423 workaround to the Cortex-A73 CPU. This needs to be
enabled only for revision <= r0p1 of the CPU.
