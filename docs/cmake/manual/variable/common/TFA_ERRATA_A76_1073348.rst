TFA_ERRATA_A76_1073348
======================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_A76_1073348

This applies errata 1073348 workaround to the Cortex-A76 CPU. This needs to
be enabled only for revision <= r1p0 of the CPU.
