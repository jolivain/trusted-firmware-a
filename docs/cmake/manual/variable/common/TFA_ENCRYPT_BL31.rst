TFA_ENCRYPT_BL31
================

.. default-domain:: cmake

.. variable:: TFA_ENCRYPT_BL31

Enables encryption of the BL31 firmware image.

Disabled by default. Forcibly disabled if :variable:`TFA_DECRYPTION_SUPPORT` is
``None``.
