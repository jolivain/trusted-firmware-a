TFA_ERRATA_A55_778703
=====================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_A55_778703

This applies errata 778703 workaround to the Cortex-A55 CPU. This needs to be
enabled only for revision r0p0 of the CPU.
