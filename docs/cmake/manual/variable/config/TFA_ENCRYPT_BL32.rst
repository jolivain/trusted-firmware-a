TFA_ENCRYPT_BL32
================

.. default-domain:: cmake

.. variable:: TFA_ENCRYPT_BL32

Enables encryption of the Secure BL32 firmware image.

Disabled by default. Forcibly disabled if :variable:`TFA_DECRYPTION_SUPPORT` is
``None``.
