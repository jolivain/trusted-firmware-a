TFA_ERRATA_A76_1791580
======================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_A76_1791580

This applies errata 1791580 workaround to the Cortex-A76 CPU. This needs to
be enabled only for revision <= r4p0 of the CPU.
