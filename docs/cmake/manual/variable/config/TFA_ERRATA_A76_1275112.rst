TFA_ERRATA_A76_1275112
======================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_A76_1275112

This applies errata 1275112 workaround to the Cortex-A76 CPU. This needs to
be enabled only for revision <= r3p0 of the CPU.
