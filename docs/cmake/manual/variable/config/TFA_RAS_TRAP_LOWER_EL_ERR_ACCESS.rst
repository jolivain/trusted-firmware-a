TFA_RAS_TRAP_LOWER_EL_ERR_ACCESS
================================

.. default-domain:: cmake

.. variable:: TFA_RAS_TRAP_LOWER_EL_ERR_ACCESS

    Enables the ``SCR_EL3.TERR`` bit to trap access to the RAS ``ERR`` and RAS
    ``ERX`` registers from lower ELs.

    Disabled by default.
