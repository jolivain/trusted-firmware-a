TFA_ERRATA_N1_1207823
=====================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_N1_1207823

This applies errata 1207823 workaround to the Neoverse-N1 CPU. This needs to
be enabled only for revision <= r2p0 of the CPU.
