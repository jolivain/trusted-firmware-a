TFA_ERRATA_N1_1868343
=====================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_N1_1868343

Applies errata 1868343 workaround to the Neoverse-N1 CPU. This needs to
be enabled only for revision <= r4p0 of the CPU.
