TFA_ERRATA_A55_846532
=====================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_A55_846532

This applies errata 846532 workaround to the Cortex-A55 CPU. This needs to be
enabled only for revision <= r0p1 of the CPU.
