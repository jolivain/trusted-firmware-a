TFA_ERRATA_N1_1073348
=====================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_N1_1073348

This applies errata 1073348 workaround to the Neoverse-N1 CPU. This needs to
be enabled only for revision r0p0 and r1p0 of the CPU.
