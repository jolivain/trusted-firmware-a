TFA_ERRATA_A73_852427
=====================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_A73_852427

This applies errata 852427 workaround to the Cortex-A73 CPU. This needs to be
enabled only for revision r0p0 of the CPU.
