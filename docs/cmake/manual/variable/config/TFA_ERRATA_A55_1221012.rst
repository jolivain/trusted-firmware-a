TFA_ERRATA_A55_1221012
======================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_A55_1221012

This applies errata 1221012 workaround to the Cortex-A55 CPU. This needs to
be enabled only for revision <= r1p0 of the CPU.
