TFA_ERRATA_A57_817169
=====================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_A57_817169

This applies errata 817169 workaround to the Cortex-A57 CPU. This needs to be
enabled only for revision <= r0p1 of the CPU.
