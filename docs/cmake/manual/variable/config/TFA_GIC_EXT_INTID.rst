TFA_GIC_EXT_INTID
=================

.. default-domain:: cmake

.. variable:: TFA_GIC_EXT_INTID

When enabled, the GICv3 driver will support extended PPI (1056-1119) and SPI
(4096-5119) ranges.

Disabled by default.
