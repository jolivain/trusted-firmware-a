TFA_ENABLE_PIE
==============

.. default-domain:: cmake

.. variable:: TFA_ENABLE_PIE

Enables Position Independent Executable (PIE) support within generic code in
|TF-A|. This option is currently only supported in BL2 when running in EL3,
BL31, and BL32 (|TSP|).

Disabled by default.
