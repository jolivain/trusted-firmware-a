TFA_ERRATA_A53_827319
============================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_A53_827319

This applies errata 827319 workaround to all CPUs. This needs to be enabled
only for revision <= r0p2 of the Cortex-A53.
