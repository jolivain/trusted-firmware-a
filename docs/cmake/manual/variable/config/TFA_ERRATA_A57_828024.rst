TFA_ERRATA_A57_828024
=====================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_A57_828024

This applies errata 828024 workaround to the Cortex-A57 CPU. This needs to be
enabled only for revision <= r1p1 of the CPU.
