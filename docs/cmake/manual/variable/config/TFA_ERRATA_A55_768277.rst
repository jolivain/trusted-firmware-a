TFA_ERRATA_A55_768277
=====================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_A55_768277

This applies errata 768277 workaround to the Cortex-A55 CPU. This needs to be
enabled only for revision r0p0 of the CPU.
