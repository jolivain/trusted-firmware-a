TFA_ERRATA_A76_1868343
======================

.. default-domain:: cmake

.. variable:: TFA_ERRATA_A76_1868343

This applies errata 1868343 workaround to the Cortex-A76 CPU. This needs to
be enabled only for revision <= r4p0 of the CPU.
